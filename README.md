# scientific-text-summarization

AI-powered system for summarizing and simplifying scientific articles to improve accessibility for non-academic audiences.

## Features
- PDF text extraction
- Automatic summarization
- Language simplification
- LLM-based simplification with controlled prompts
- Structured output with key concept explanations

## Tech Stack
- Python
- NLP / LLMs
- PDF Processing
- LLM API (planned)

## 🎯 Goal
Democratize access to scientific knowledge by translating complex content into accessible language.

## 📌 Status
In development – initial pipeline completed.
